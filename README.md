# SPACE SHOOTER

## Description
I Created this game to experience the og space shooter game which was very cool. This Project helped me in learning some new things and Improving my skills in godot.
It has some old game features like enemies shooting bullets,bouncing enemies shooting bullets Meteors etc.
It also have some Power Ups for Defense and Attack Boost.
The Structure of the Project is very SImple and Alegant
I will try to add more levels in the future.
I have also added Game Over Screen Which Pops up when Player Dies
